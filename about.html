GERENCIADOR DE TAREFAS

Sobre o projeto

Este é um projeto de gerenciamento de tarefas desenvolvido em dupla. O objetivo é fornecer uma interface simples e intuitiva para gerenciar suas tarefas diárias. Você pode adicionar, editar e excluir tarefas, além de marcar as concluídas.

O projeto foi desenvolvido utilizando HTML, CSS e JavaScript, com foco na responsividade e usabilidade. A ideia é que você possa acessar suas tarefas de qualquer dispositivo, seja um computador, tablet ou smartphone.

Além disso, a API utilizada para o gerenciamento de usuários e tarefas foi construída com Node.js e Express, garantindo uma comunicação eficiente entre o front-end e o back-end. A autenticação é feita através de tokens JWT, proporcionando segurança no acesso às funcionalidades.

Por fim, esse projeto foi construído com o intuito de praticar e aprimorar habilidades em desenvolvimento web, além de ser uma ferramenta útil para o dia a dia. Espero que você goste e encontre valor na aplicação!
